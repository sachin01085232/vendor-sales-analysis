# 📊 Vendor & Brand Performance Analysis

## 🚀 Project Overview
This project analyzes vendor purchases, sales performance, freight cost, and pricing data using **PostgreSQL & Python**.

The objective is to transform raw transactional data into an **analytics-ready dataset** and generate profitability insights for better business decision-making.

- PostgreSQL
- SQL
- Python
- Pandas

## ⚙️ Project Workflow

### 1️⃣ Data Extraction (SQL)
- Extracted and combined data from multiple tables
- Used **JOINs, CTEs, and aggregations**
- Created a final vendor summary table

### 2️⃣ Data Cleaning (Python)
- Handled missing values
- Fixed inconsistent data types
- Removed unwanted spaces and null records

### 3️⃣ Feature Engineering
Created new business metrics:
- Gross Profit
- Profit Margin
- Stock Turnover
- Sales to Purchase Ratio

---

## 📊 Business Insights
- Top vendors contribute the majority of total sales
- High purchase quantity does not always result in high sales
- Freight cost significantly reduces profitability for low-volume vendors
- Some products were purchased at a higher price than the actual listed price
